# 🚀 Gen AI Query Engine

### 🛠️ **Project Overview**
The **Gen AI Query Engine** is a lightweight backend service that simulates a Gen AI-powered data query system.  
It allows users to:
- Execute natural language queries and convert them to SQL-like statements.
- Get SQL breakdowns from natural language queries.
- Validate SQL query feasibility.
- Use authentication with API keys.

---

## ✅ **Tech Stack**
- **Backend:** Node.js + Express.js
- **Database:** SQLite (in-memory)
- **Authentication:** API Key
- **Deployment:** Render/Railway/Heroku
